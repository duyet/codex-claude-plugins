# HTML Artifact Page Types

The `anthropic-html` skill supports several specialized page types, each optimized for different engineering and research tasks.

## 1. Exploration & Comparison
*   **Code Approaches:** Side-by-side view of multiple implementation options. Uses 2 or 3-column grids.
*   **Visual Designs:** Renders live UI mockups or layout options for fast feedback.
*   **Component Variants:** Shows the same component with different "vibes" or states.

## 2. Engineering Workflows
*   **PR Review:** A dashboard showing diffs, finding severity, and architectural context.
*   **Code Understanding:** Explains complex logic using inline annotations and split views.
*   **Implementation Plan:** Visual timeline of milestones, risk matrices, and data flow.
*   **PR Writeup:** Professional summary of changes with "before/after" screenshots (rendered in HTML).

## 3. Reporting & Triage
*   **Status Report:** High-level overview of project health, blocking issues, and next steps.
*   **Incident Report:** Timeline of events, root cause analysis, and impact assessment.
*   **Triage Board:** Kanban-style view for managing bugs or feature requests.
*   **Feature Flags / Prompt Tuner:** Interactive dashboards for adjusting configuration or tuning LLM prompts.

## 4. Research & Knowledge
*   **Concept Explainer:** Deep dive into a technical topic using interactive "explodables" or tabs.
*   **Feature Explainer:** Walkthrough of a specific feature's logic and data flow.
*   **Slide Deck:** A presentation format using full-screen sections and navigation.

## 5. Visualizations
*   **SVG Flowcharts:** Data flow or logic diagrams that are searchable and interactive.
*   **SVG Illustrations:** Technical diagrams (e.g., cloud architecture, memory layout).
