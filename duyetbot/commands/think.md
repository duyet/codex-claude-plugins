# Duyetbot Think

Deep structured analysis for: `$ARGUMENTS`

## Framework

### 1. Problem Definition
- Core problem?
- Symptoms vs root causes?
- Constraints?

### 2. Context Analysis
- Existing patterns in codebase?
- What's been tried?
- Dependencies?

### 3. Solution Design
- Possible approaches?
- Tradeoffs?
- Best fit?

### 4. Execution Plan
- Exact steps?
- What could fail?
- Verification method?

## Output

Structure findings as:

```
## Problem
[Core issue]

## Analysis
[1] Finding → Implication
[2] Finding → Implication

## Approach
[Recommended solution with rationale]

## Plan
1. Step one
2. Step two
...

─── duyetbot ── thinking ─────
```

Use Sequential Thinking MCP for complex multi-step reasoning if available.
